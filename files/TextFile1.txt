ERROR code doc
Hopefully this file helps to trooubleshoot

ERROR-noDataTxt - data.txt was not found to exist in the expected location, locate the file and return it or restore from a back-up or overwrite the file
ERROR-missingFile - A file that contains data key to the progam is missing, please reinstate
ERROR-badParameter - A function was passed data it did not expect or understand (eg, it was expecting "y" or "n" but was passed "p)
ERROR-fileCheckFail - The file check made it to the end of the checks and encountered an unknow error, reloading the program should fix this, if not, contact support
